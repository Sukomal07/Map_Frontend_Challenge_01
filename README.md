# Map_Frontend_Challenge_01

## Installation

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/Sukomal07/Map_Frontend_Challenge_01.git
   cd Map_Frontend_Challenge_01
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run Project:

   ```bash
   npm run dev
   ```
